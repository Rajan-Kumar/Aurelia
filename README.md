# Aurelia

Configure Through Visual Studio Editor

1) Open a blank Web Application.
2) Install Node
3) Install npm  
4) Install jspm (npm install jspm -g)
5) Navigate to the root of the project in command prompt.
6) Hit Command: jspm init. To generate the jspm_module folder and to generate the system.js & config.js.
7) Add codex page. Add system.js reference and config.js reference and import aurelia-bootstrapper through System.import('aurelia-bootstrapper');'
and decorate body with aurelia-app.
8) Add app.js and app.html page in the project.
9) Install aurelia-framework & aurelia-bootstrapper.
10) Launch index.html page in brower.
